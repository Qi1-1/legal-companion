# 法律陪伴 · Legal Companion

> **你的事,你有权搞明白。**
> 把这几行字,粘进你已经在用的任何 AI(豆包、DeepSeek、文心、Kimi…),它立刻变成一个**站在你这边、说人话**的法律帮手——不绕术语、不端架子,陪你把憋屈的事一步步理清楚。
>
> 永远免费,永远开源。**因为懂法,不该是有钱人的特权。**

## 凭什么用它
- **它是为你做的,不是为律师做的。** 市面上的法律 AI 大多在伺候专业人士;这一个,只为那个刚撞上墙、求告无门的普通人。
- **它说人话。** 你一个法律词都不用懂,用大白话讲你的事就行,它用大白话回你。
- **它站在你这边。** 被欠薪、被骗、被强拆、起纠纷——它帮你看清:这事是什么、你占不占理、有哪些路、第一步今天就能做什么。
- **想做事也能问它。** 开店、辞职、签合同、借钱、租房——怎么做才合法、怎么不踩坑,它当你的合规副驾。

## 三步,现在就能用
1. 打开 [`skills/legal-companion/SKILL.md`](skills/legal-companion/SKILL.md),**全选复制**。
2. 粘进你的 AI,发送。
3. 用大白话把你的事讲给它听。

## 它的底线,也是它对你的承诺
它给的是看得懂、用得上的法律思路。**真该找律师的时候,它会明明白白告诉你,绝不糊弄。** 遇到大事、急事,它会把你领到执业律师和**免费的全国法律援助热线 12348**——这不是免责,是它对你负责。
