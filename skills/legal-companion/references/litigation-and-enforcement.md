# 自己打民事官司 + 强制执行(照着做版)

诉讼费、小额诉讼标的额、网上立案细节各地有差异,标"需核实"的以当地法院当年公告为准。

## 官方平台(先收藏)
- **人民法院在线服务**:电脑端 `zxfw.court.gov.cn`;手机端微信小程序"人民法院在线服务全国版"。网上立案、调解、执行、保全一站办。(原"移动微法院"多已并入)
- **中国执行信息公开网** `zxgk.court.gov.cn`:查被执行人、失信名单、限高、终本、执行悬赏。

## 网上立案(电脑/手机通用)
注册实名(身份证+短信+人脸)→ 预留电子签名 → 选"当事人" → "我要立案"(审判立案 / 执行立案)→ 选有管辖权的法院 → 选案由 → 填信息+传材料(起诉状Word、身份证明、证据)→ 签《送达地址确认书》→ 提交。法院短信通知审核结果。

## 选对通道
- **支付令**(欠款清楚、对方只是拖时最快最便宜):走督促程序、不开庭。条件:给付金钱/有价证券、债权到期且数额确定、与对方无其他纠纷、能送达。向**基层法院**申请,受理后15日内发令。申请费按财产案件受理费的**1/3**。**最大坑:对方收到15日内提书面异议(无需说理)→ 支付令失效,得另行起诉**。对方不异议又不还→直接申请执行。
- **小额诉讼**(标的小、事实清楚):基层法院,**一审终审、不能上诉**;标的额按各省上年度就业人员年平均工资比例划线(**各省不同,需核实**);诉讼费减半(部分地区更低,如北京每件10元);约2个月内审结。
- **简易程序**约3个月、**普通程序**6个月;不服一审15日内上诉。

## 诉讼费 + 缓减免
财产案件受理费分段累计:**≤1万每件50元**;1万–10万 2.5%;10万–20万 2%;20万–50万 1.5%……(简易/小额减半,原则由败诉方负担)。
**交费困难可申请缓/减/免**:追索社保金/经济补偿、交通医疗工伤等人身损害受害人、正在受法律援助的**应准予缓交**;低保/受灾困难等可减免。立案时交司法救助申请书+困难证明(低保证、村居委证明等)。

## 强制执行
- **申请执行时效2年**(从生效文书规定的履行期最后一日起算),别拖过。
- 在"人民法院在线服务"→执行立案,传执行申请书+生效文书+身份材料+证据;法院应7日内立案。
- **法院的武器**:网络查控("总对总")查冻存款/不动产/车辆/股票/第三方支付(支付宝、微信);查封拍卖财产;**限制高消费9类**(飞机、高铁一等座、星级酒店、买房买车、子女高收费私立学校…);**纳入失信名单(老赖)**;罚款、拘留直至拒执罪。

## "执行难"——对方藏财产时什么真有用
1. **自己提供财产线索**:对方银行账号、房产车辆、工资公积金、第三方支付账户、上下游/客户处的资金货物去向——书面交执行法官请求查控。
2. **执行悬赏公告**:申请法院发悬赏,从执行到的财产里按比例付;多地有"悬赏保险"降低垫付风险。
3. **追加/变更被执行人**:股东抽逃出资/出资不实、一人公司财产混同等法定情形,可把"背后的人/钱"拉进来(须有明文依据)。
4. **拒执罪自诉**:有能力执行而拒不执行、情节严重(《刑法》第313条),公安检察不立案时可备刑事自诉状+证据向执行法院所在地法院提自诉——逼老赖还钱的强力杠杆。
5. **"终本"≠案子结束**:查不到财产时法院会"终结本次执行程序(终本)",但这不是终点——以后**发现财产线索可随时申请恢复执行,且不再受2年时效限制**。拿到终本裁定别以为钱彻底没了。

## 一句话决策树
欠款清楚、对方只是拖 → **支付令**;怕异议或标的小 → **小额诉讼**;标的大/有争议 → 简易或普通程序起诉,全程网上办 → 拿到生效文书对方不还 → **2年内执行立案 + 交财产线索 + 申请限高/失信** → 查不到财产 → 悬赏/追加被执行人 → 明显转移财产 → **拒执罪自诉**。

## 需核实
各省小额诉讼标的额与诉讼费优惠;当地是否保留独立"移动微法院"入口。
